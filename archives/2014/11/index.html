Will Learning to Code Empower Me?

Coming from a mathematical background, I’ve spent years building mathematical models to describe the behavior of a system. These systems have ranged from statistics, to biology, and computer science. I’ve used modeling software, including Matlab, to help get a hands on visual model of a system. While doing this, I have realized how much there is to learning about coding.

Matthew Kirschenbaum has written about learning to code in order to understand computers as “engines for creating powerful and persuasive models of the world around us.” I feel that my getting a stronger understanding and capability of coding, it will make me a stronger applied mathematician.

By definition the word empower means to make someone stronger and more confident. By being empowered, we learn more, experience more, and can do more. I look forward to how learning to code is going to empower me.
